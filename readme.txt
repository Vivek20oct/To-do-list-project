To-Do List Application
======================

This is a simple console-based To-Do List application created using Python.  
You can add tasks, remove tasks, and view your to-do list easily.  

Features:
---------
1. Add a task to the list.
2. Remove a task from the list.
3. View all tasks in the list.
4. Quit the application.

How to Run:
-----------
1. Install Python 3 on your system.
2. Run the script using the command:
   python todo_list.py
3. Follow the instructions on the screen to use the app.

Example:
--------
- Add a task: "Buy groceries"
- View your list:
  1. Buy groceries
- Remove the task: "Buy groceries"

Enjoy managing your tasks with this simple To-Do List app!
